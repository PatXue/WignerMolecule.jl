\section{Ferromagnet}
For the ferromagnet, temperature and energy are in units of meV, and order
parameters are the spin expectation (magnetization) and $\eta_z$ expectation.
\subsection{Zero-Field}
\begin{itemize}
    \item State is ordered below $T=4.5$ meV, when initialized in ground state (high bias field),
    also see heat capacity spike at that temperature
    \item Transition temperature does not seem affected by system size
\end{itemize}

    \includegraphics[width=0.75\textwidth]{fm_heatcap.png}
\end{center}

\subsection{Finite Bias Field}
\begin{itemize}
    \item Applying a constant bias field on the spins
    \item Swept temperature below, at, and above transition (4.5, 5.5, 6.5 meV)
    \item Varied system size between 20$\times$20, 40$\times$40, and 80$\times$80
    \item At all system sizes, below transition temperature, $\eta_z$ is flat for
    low fields
\end{itemize}

\subsection{Next Steps}

    \item Try sweeping AFM-FE parameter space
\end{itemize}
\subsection{Result}
\begin{itemize}
    \item Orders weakly in annealed runs with no bias
    \item With bias field, system seems to transition to phase at some finite bias field, doesn’t change with system size
    \item Increasing temperature moves transition to higher field
    \item Eta-only runs show strong ordering up to very high temperatures
\end{itemize}
